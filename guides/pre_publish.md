# Pre-Publish Checklist — Sleep Channel

Run through every item before clicking "Publish". This protects monetization.

---

## FICHIER FINAL

- [ ] Fichier MP4 dans `output/` confirmé
- [ ] Durée vérifiée : **10h00 minimum** (lecture dans VLC ou propriétés Windows)
- [ ] Taille fichier : ~3-8 GB selon la vidéo source (normal)
- [ ] Aucun artefact visuel sur les premiers/derniers 10 secondes

---

## DROITS & SOURCES

- [ ] Vidéo source : téléchargée depuis **Pixabay** (licence libre commerciale, pas de CC-BY requis)
- [ ] Audio source : vérifié libre de droits (Pixabay Audio, Freesound.org CC0, ou Epidemic Sound avec safelist)
- [ ] Aucun son de musique : sons naturels **purs** seulement (feu, pluie, forêt)
- [ ] Aucun élément protégé par Content ID non résolu

---

## CONTENU YOUTUBE

- [ ] **Titre** : Format storytelling copié depuis `content/titles.txt`
  - Mot-clé principal EN PREMIER
  - Univers unique (pas de titre générique "Rain Sounds for Sleep")
  - < 100 caractères
- [ ] **Description** : Template copié depuis `content/descriptions.txt`
  - Hook visible ligne 1 (avant "Show more")
  - Chapters / timestamps inclus
  - CTA abonnement présent
  - Hashtags inclus (max 15)
- [ ] **Tags** : 25-35 tags depuis `content/tags.txt` (broad + niche + long-tail)
- [ ] **Thumbnail** : Uploadée manuellement depuis Canva (1280×720 px minimum)
  - Style sombre, texte blanc bold, image forte
  - Différente visuellement des vidéos précédentes
- [ ] **Chapters** : Template copié depuis `content/chapters.txt`

---

## CONFIGURATION YOUTUBE STUDIO

- [ ] Catégorie : **Music** (pas "Entertainment")
- [ ] Langue : **English (United States)**
- [ ] Monétisation : **activée AVANT publication**
  - Ad formats : Skippable ads + Non-skippable ads + Bumper ads
  - **Désactiver les mid-rolls** (ruinent l'expérience sleep sur 10h)
  - Activer les overlay ads seulement
- [ ] Playlist : assignée à la bonne playlist thème (crée si manquante)
  - Exemples: "Fireplace Ambience", "Rain Sounds", "Forest Night Sounds"
- [ ] Mode : Public (ou Scheduled pour publier dans 24h)

---

## ANTI-DÉMONÉTISATION (règles juillet 2025)

- [ ] Univers storytelling **unique** pour cette vidéo (pas une copie de la précédente)
- [ ] Titre différent du pattern exact d'une autre vidéo sur la chaîne
- [ ] Thumbnail visuellement distincte des vidéos précédentes
- [ ] Audio différent de la vidéo précédente (pas même fichier source rebouclé)
- [ ] Aucune musique en fond (Content ID risk)

---

## POST-PUBLICATION (première heure)

- [ ] Répondre à **tous** les commentaires dans l'heure
  - Ex: "Glad you're here! Let us know your favorite sleep universe 🔥"
- [ ] Partager sur **Pinterest** (créer une épingle avec la thumbnail)
- [ ] Partager sur **Reddit** : r/ambientmusic, r/sleep, r/relaxation
  - Titre Reddit: "I made a 10-hour [UNIVERSE] ambience for sleep — free, no ads"
- [ ] Tweeter la vidéo avec le titre + hashtags
- [ ] Vérifier dans YouTube Analytics 30 min après: CTR > 2% = bon signal

---

## NOTES
- Ne jamais cliquer "Save & Publish" avant d'avoir tout coché
- La monétisation activée AVANT publication = vidéo monétisée dès la 1ère vue
- Première vidéo : attends 2-3 jours avant de juger les métriques (délai d'indexation)
